# 🚀 Quick Start Guide - Arizona Budget Dashboard

## Get Started in 3 Easy Steps!

### Step 1: Open the Dashboard
Simply **double-click** the `index.html` file in the `arizona-budget-dashboard` folder.

The dashboard will open in your default web browser - **no installation required!**

### Step 2: Explore the Dashboard

#### Home View
- See all 6 Arizona departments at a glance
- Each card shows:
  - Department icon and name
  - FY 2027 budget
  - Change from FY 2026
  - Up/down arrow indicator

#### Click Any Department Card
- View detailed budget information
- See interactive charts
- Compare FY 2026 vs FY 2027
- Read significant changes

### Step 3: Use the Features

#### Navigation Bar (Top)
- **All Departments**: Return to home view
- **ADOA, DES, DCS, DEQ, DOC, AHCCCS**: Jump to specific department

#### Department Detail View
- **Budget Overview**: Three cards showing FY26, FY27, and changes
- **Funding Sources Chart**: Bar chart comparing funding sources
- **Category Breakdown**: Pie chart showing budget categories
- **Significant Changes**: List of major changes
- **Export Data**: Download CSV file
- **Print View**: Print-friendly layout

#### Keyboard Shortcuts
- **Escape**: Return to dashboard
- **Left/Right Arrows**: Navigate between departments

## 📊 The 6 Departments

1. **ADOA** 🏛️ - Arizona Department of Administration
   - Administrative support services
   - ~$132M budget (FY27)

2. **DES** 🤝 - Department of Economic Security
   - Social services and economic assistance
   - ~$3.1B budget (FY27)

3. **DCS** 👨‍👩‍👧‍👦 - Department of Child Safety
   - Child protection services
   - ~$845M budget (FY27)

4. **DEQ** 🌱 - Department of Environmental Quality
   - Environmental protection
   - ~$102M budget (FY27)

5. **DOC** ⚖️ - Department of Corrections
   - State prison system
   - ~$1.32B budget (FY27)

6. **AHCCCS** 🏥 - Arizona Health Care Cost Containment System
   - Arizona's Medicaid program
   - ~$16.8B budget (FY27)

## 💡 Tips

### Best Experience
- Use a modern browser (Chrome, Firefox, Safari, Edge)
- Desktop or tablet recommended for full features
- Mobile-friendly but charts are better on larger screens

### Exporting Data
1. Click on any department
2. Scroll to bottom
3. Click "📊 Export Data (CSV)"
4. File downloads automatically

### Printing
1. Click on any department
2. Click "🖨️ Print View" button, or
3. Use browser's print function (Ctrl+P / Cmd+P)

### Navigation
- Click department cards to view details
- Use top navigation bar to switch departments
- Click "← Back to Dashboard" to return home
- Use keyboard arrows to navigate between departments

## ❓ Need Help?

### Dashboard Not Loading?
- Make sure you opened `index.html` in a web browser
- Try a different browser
- Check that all files are in the same folder

### Charts Not Showing?
- Check your internet connection (Chart.js loads from CDN)
- Try refreshing the page (F5 or Ctrl+R)
- Clear browser cache

### Data Looks Wrong?
- The current data is sample/placeholder data
- To update with real data, edit `data.js`
- See README.md for detailed instructions

## 📝 Next Steps

### To Update Budget Data
1. Open `data.js` in a text editor
2. Update the numbers for each department
3. Save the file
4. Refresh the dashboard in your browser

### To Customize Colors
1. Open `styles.css` in a text editor
2. Find the `:root` section at the top
3. Change the color values
4. Save and refresh

### To Add More Departments
1. Add department data to `data.js`
2. Add navigation button in `index.html`
3. Save and refresh
